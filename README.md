  ## ✨ Features

  ### Core Features (Same as Stable Version)
  - ✅ Real-time group chat and private messaging
  - ✅ Friend system (auto-accept friend requests)
  - ✅ Modern GUI interface
  - ✅ User authentication (SHA256 password hashing)
  - ✅ Bilingual support (English/Chinese)
  - ✅ Cross-platform (Windows, macOS, Linux)

  ### Experimental Features
  - 🧪 **chat_client_simple.py** - Simplified file upload client (in testing)

  ### System Requirements
  - Python 3.8 or higher

  **Windows:**
  ```bash
  install.bat

  macOS/Linux:

  ./install.sh

  ---
  📖 Installation

  Manual Installation

  1. Clone Repository
  git clone https://github.com/lukeqiang2000/lan-chat-python.git
  cd lan-chat-python

  2. Install Dependencies
  pip install -r requirements.txt

  3. Run Server
  python chat_server.py

  4. Run Client
  python chat_gui.py

  ---
  🎮 Running the Project

  Start Server

  python chat_server.py
  Server runs on port 9999 by default

  Start Client

  # GUI Client (Recommended)
  python chat_gui.py

  # Simple Client (Beta)
  python chat_client_simple.py
